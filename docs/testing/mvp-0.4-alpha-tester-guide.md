# 덜어냄 MVP 0.4-alpha 테스터 가이드

## 설치 방법

### 방법 1: 인스톨러 사용 (권장)
1. `DeoReoNem-0.4-alpha-setup.exe` 실행
2. 설치 경로 선택 (기본: `C:\Program Files\ScopeWorks\DeoReoNem`)
3. 설치 완료 후 시작 메뉴 또는 바탕화면에서 실행

### 방법 2: ZIP 압축 해제
1. ZIP 파일 압축 해제
2. `deoreonem_desktop.exe` 직접 실행

## 테스트 체크리스트

### 기본 흐름
- [ ] 앱 실행 → 시작하기 → 텍스트 입력 (줄마다 하나씩)
- [ ] 분류하기 → 각 항목에 카테고리 지정
- [ ] 첫 번째 행동 선택
- [ ] 요약 화면 확인
- [ ] 완료 후 시작 화면 복귀

### 리뷰 기능
- [ ] "맡겨둔 것 확인하기" 버튼 표시 확인
- [ ] 일정 / 메모 / 감정 서랍 탭 전환
- [ ] 걱정 항목 "이 걱정 내려놓기" → 양분 변환 메시지 확인
- [ ] "다시 3일 맡겨두기" 동작 확인

### 작은 자리 (가든 오버레이)
- [ ] "작은 자리 보기" 클릭 → 오버레이 창 열림
- [ ] 오버레이 드래그 이동
- [ ] 오버레이 더블클릭 → 메인 앱 열기
- [ ] Esc로 오버레이 닫기
- [ ] 걱정 내려놓기 후 → 오버레이에 미세한 빛남 효과

### 중복 실행 방지
- [ ] 메인 앱 실행 중 → 다시 실행 시 자동 종료
- [ ] 작은 자리 열려 있을 때 → "작은 자리 보기" 클릭 시 안내 메시지
- [ ] "작은 자리 보기" 빠르게 연타 → 하나만 열림

### 다중 세션 데이터 보존
- [ ] 첫 번째 세션 완료 → 두 번째 세션 완료 → 리뷰에서 두 세션의 항목 모두 표시
- [ ] 앱 종료 후 재실행 → 이전 세션 데이터 유지

### 한글 입력
- [ ] 텍스트 입력 시 한글 조합 중 충돌 없음
- [ ] 여러 줄 입력 후 분류하기 정상 동작

## 런타임 데이터 위치

앱 런타임 데이터 저장 경로:
```
%APPDATA%\ScopeWorks\DeoReoNem
```

이 폴더에는 다음 파일이 저장됩니다:
- `completed_sessions.json` — 최근 완료 세션 ID
- `garden_state.json` — 작은 자리 양분 상태
- `main_app.lock` / `garden_overlay.lock` — 중복 실행 방지 잠금

## 초기화 방법

모든 로컬 데이터를 초기화하려면:
1. 앱 완전 종료 (작은 자리 포함)
2. `%APPDATA%\ScopeWorks\DeoReoNem` 폴더 삭제
3. 앱 재시작
